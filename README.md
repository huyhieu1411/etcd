# Summary
The UI management for etcd
# Features
* Connection manager
* Key - value manager
* User management
* Role management
* User - Role permission
* Import / Export keys
* View list revision / Create new revision from old revision
* Compare diff beetwen revision
# Library
* Angular 14
* Primeng 13
* Asp.NET Core 6
* LiteDb
* Sakai theme free: https://www.primefaces.org/sakai-ng/
# Development

## App
Should use pnpm:
```code
npm i -g pnpm
```
then:
```code
cd app
pnpm install
pnpm run start
```

## API
```code
cd api/EtcdManager.API
dotnet watch
```
